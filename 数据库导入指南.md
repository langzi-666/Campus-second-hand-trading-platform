# 数据库导入指南

## 方法一：使用MySQL命令行

1. 打开MySQL命令行客户端
```bash
mysql -u root -p
```

2. 输入密码后，执行以下命令：
```sql
SOURCE database/init.sql;
```

或者：
```sql
-- 创建数据库
CREATE DATABASE IF NOT EXISTS campus_trade DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 使用数据库
USE campus_trade;

-- 导入表结构和数据
SOURCE database/init.sql;
```

## 方法二：使用批处理文件

双击运行项目根目录下的 `import_db.bat` 文件，按提示输入MySQL密码。

## 方法三：使用PowerShell

在项目根目录下打开PowerShell，执行：
```powershell
Get-Content database/init.sql | mysql -u root -p
```

## 验证导入结果

导入完成后，可以通过以下SQL验证：

```sql
-- 查看数据库
SHOW DATABASES;

-- 使用数据库
USE campus_trade;

-- 查看表
SHOW TABLES;

-- 查看用户数据
SELECT * FROM users;

-- 查看分类数据
SELECT * FROM categories;

-- 查看商品数据
SELECT * FROM products;
```

## 注意事项

1. 确保MySQL服务已启动
2. 确保用户名和密码正确
3. 如果遇到权限问题，请联系数据库管理员
4. 导入前请备份现有数据（如有）
